1. app/은 React 앱의 전체 설정과 시작 구조를 담당합니다.
2. pages/는 URL에 직접 연결되는 화면입니다.
3. layouts/는 여러 페이지가 공유하는 화면 틀입니다.
4. features는 기능 단위 코드를 넣는 곳입니다.
- features/auth/api/는 로그인 API 요청을 담당합니다.
- features/auth/components/는 로그인 기능 전용 컴포넌트가 들어갑니다.
- features/auth/contexts/는 앱 전체에서 로그인 정보를 공유하는 저장소입니다.
